CreateAppX

CreateAppX will generate valid appx for windows 8.1 Universal Projects. It also allows to update manifest for projects.

About CreateAppX

CreateAppx is a tool created to update, build and deploy Windows 8.1 universal project. It allows users to manually update manifest file and build project without use of visual studio. It also validates build folder and generates final .appx file can be directly uploaded to windows store. This tool is created specially for 'Appzillon' users.

Requirements

You will need Windows 8.1 SDK.
Build folder of Windows 8.1 generated by Appzillon 3.1 IDE.
